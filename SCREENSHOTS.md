# Rustmix X4 Screenshots

This document provides screenshot references for the first Rustmix X4 firmware release. The images show the main UI categories, reader workflows, apps, network setup, system settings, and sleep-image behavior on the Xteink X4 device.

Use these screenshots in the README, user guide, GitHub release page, or project website. The images are stored in `screenshots/` and are optimized for GitHub display.
