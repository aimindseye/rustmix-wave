# Rustmix Wave screenshots

This directory contains the reference UI screenshots used by [`docs/USER_GUIDE.md`](../docs/USER_GUIDE.md).

The images document the physically verified Rustmix Wave v1.0.0 screen surfaces and the v1.1.0 Devanagari and Gujarati EPUB Reader surfaces. They are intentionally kept as repository assets so the user guide can use stable relative links on GitHub.

The screenshot filenames are descriptive and map directly to the guide sections. Runtime firmware does not read files from this directory.

## v1.1.0 Indic EPUB screenshots

- `epub-devanagari-bhagavat.jpg`: large Hindi / Sanskrit Bhagavat EPUB rendered through Noto Sans Devanagari.
- `epub-devanagari-garud-puran.jpg`: Hindi Garud Puran EPUB rendered through Noto Sans Devanagari.
- `epub-gujarati-valmiki-ramayan.jpg`: Gujarati Valmiki Ramayan EPUB rendered through Noto Sans Gujarati.

See [`../Screenshots.md`](../Screenshots.md) for the physical-verification gallery.
