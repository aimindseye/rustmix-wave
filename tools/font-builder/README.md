# Rustmix Wave Indic font-pack builder

Open `index.html` in a modern browser. The builder runs locally and does not upload font files or EPUB text.

1. Obtain `NotoSansDevanagari-Regular.ttf` and `NotoSansGujarati-Regular.ttf` from the official Noto projects.
2. Extract a corpus from the EPUB files:

   ```bash
   python3 scripts/extract-epub-font-corpus.py book1.epub book2.epub > indic-corpus.txt
   ```

3. Select the local font files and the extracted corpus in the browser tool.
4. The builder downloads one `rustmix-indic-font-pack.zip` archive. Extract it, then copy `FONTS.TXT` and every `.RWF` file to `/RUSTMIX/FONTS` on the SD card.

The builder intentionally emits one ZIP download. Browsers may block or discard repeated automatic downloads triggered by a single click.

The packs contain shaped cluster bitmaps generated by the browser canvas. Raw TTF and OTF files are not copied to the firmware repository or the device.

## Lighter e-paper raster threshold

The builder exposes a 1-bit alpha threshold. Start with `160` when using Noto Sans Devanagari Regular and Noto Sans Gujarati Regular. Higher values discard more anti-aliased edge pixels and make the generated page lighter. Regenerate and reinstall the ZIP after changing the threshold.
