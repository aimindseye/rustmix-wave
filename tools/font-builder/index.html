Rustmix Wave Indic Font Pack Builder

This local browser tool generates SD-card Reader packs for Devanagari and Gujarati EPUB text. Font files stay in your browser and are not uploaded.

Use Noto Sans Devanagari and Noto Sans Gujarati. First extract a UTF-8 corpus from the EPUB files with scripts/extract-epub-font-corpus.py, then select that corpus below. The tool downloads one ZIP so browser multiple-download blocking cannot omit pack files.

The builder includes standalone script codepoints and corpus-specific shaped clusters so conjuncts, matras and Gujarati clusters render correctly on the device.

E-paper raster weight

Higher values discard more anti-aliased edge pixels and create a lighter 1-bit glyph. Start with 160 for Noto Sans Regular.
